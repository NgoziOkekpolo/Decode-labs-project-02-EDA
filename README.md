# Decode-labs-project-02-EDA
Exploratory Data Analysis (EDA) on a cleaned e-commerce dataset using Python

## Project Overview

This project was completed as part of the DecodeLabs Data Analytics Internship (Project 2).

The objective of this project was to perform an Exploratory Data Analysis (EDA) on a cleaned e-commerce dataset to better understand customer behaviour, sales trends, product performance, and relationships between variables. The analysis was carried out using Python to uncover meaningful insights that can support business decision-making.

---

## Project Objectives

- Explore the structure of the dataset.
- Perform data quality checks.
- Analyze the distribution of numerical and categorical variables.
- Identify relationships between variables.
- Detect potential outliers.
- Generate business insights and recommendations.

---

## Key Findings

- Instagram generated the highest number of customers and total sales.
- Facebook customers recorded the highest average order value.
- Credit card payments contributed the highest sales revenue.
- Laptops recorded the highest average order value among all products.
- Unit Price showed the strongest positive relationship with Total Price.
- Only a few high-value outliers were identified in the Total Price column.

---

## Recommendations

- Increase marketing efforts on Instagram to attract more customers.
- Investigate the reasons behind cancelled orders.
- Continue offering free shipping promotions, as they were widely used.
- Encourage credit card payments through incentives.
- Promote high-value products such as laptops.
- Monitor high-value transactions for better business oversight.
